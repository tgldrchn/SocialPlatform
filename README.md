# SocialNetworkingPlatform

A Discord-like social platform built with C# and OOP principles.

## Projects
- **SocialNetworkingPlatform** — Core DLL with reusable base classes
- **DiscordApp** — Console App that extends the DLL

## Features
- User registration and login with password hashing
- Posts, comments, and reactions
- Direct messages and group chats
- Discord servers with text and voice channels
- Role management (Owner, Admin, Moderator, Member)

## Tech
- C# .NET 10
- Abstract classes, Interfaces
- Repository and Service patterns
